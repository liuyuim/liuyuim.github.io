  • 通过 Anaconda 管理 Python 包


    Anaconda集成了常用的扩展包，能够方便地对这些扩展包进行管理，比如安装和卸载包，这些操作都需要依赖conda。conda是一个在Windows、Mac OS和Linux上运行的开源软件包管理系统和环境管理系统，可以快速地安装、运行和更新软件包及其依赖项。在Windows系统下，用户可以打开Anaconda Prompt工具，然后在Anaconda Prompt命令
    注意：如果希望快速了解如何使用conda命令管理包，则可以在Anaconda"conda-h”或"conda-help"命令来查看帮助文档。
    【检测anaconda是否被安装】
    conda -- version
    conda4.5.4
    一旦发现有，就会返回其当前的版本号。
    【查看python版本】
    python -V
    Python 3.8.8
    【环境】
    查看环境
    conda env list 或 conda info –-envs
    创建新环境
    conda create -n 环境名
    conda create -n 环境名 python=版本号
    切换环境
    conda activate 环境名
    退出环境
    conda deactivate 环境名
    删除环境
    conda remove -n 环境名 --all
    导出环境
    conda env export > 文件名.yaml
    导入环境
    conda env create -f 文件名.yaml
    【复制/克隆一个环境】
    通过克隆来复制一个环境。这儿将通过克隆old_envs来创建一个称为new_envs的副本。
    conda create -n new_envs --clone old_envs
    通过conda info –-envs来检查环境
    你现在应该可以看到一个环境列表有 new_envs, old_envs
    删除一个环境
    如果你不想要这个名为new_envs的环境，就按照如下方法移除该环境：
    conda remove -n new_envs --all
    为了确定这个名为flowers的环境已经被移除，输入以下命令：
    conda info -e
    new_envs 已经不再在你的环境列表里了，所以我们知道它被删除了。
    【清理环境tarball】
    移除索引缓存，锁定文件，未使用的缓存包和tarball。
    conda clean -a -all
    检查哪些包没有在包缓存中被硬依赖到其他地方，并删除它们
    conda clean -p --packages / conda clean -t --tarballs
    帮助
    conda clean -h
    【查看当前环境下包信息】
    使用“list"命令可以获取当前环境中已经安装的包信息，命令格式如下。
    conda list， conda list | findstr numpy
    【查找包】
    使用search命令可以查找可供安装的包，命令格式如下
    conda search --full-name 包的全名
    上述命令中，--full-name为精确査找的参数.后⾯紧跟的是包的全名。例如，查找全名为“python”的包有哪些版本可供安装，示例命令如下。conda search --full-name python
    【安装包】
    使用install命令可以安装包。如果希望在指定的环境中进行安装，则可以在install命令的后⾯显式地指定环境名称，命令格式如下。
    conda install --name env_name package_name
    上述命令中，env_name参数表示包安装的环境名称，package_name表示将要安装的包名称。
    例如，在Python 3环境中安装pandas包，示例命令如下。
    conda install --name python3 pandas
    如果要在当前的环境中安装包，则可以直接使用install命令进行安装，命令格式如下。
    conda install package_name
    指定版本
    conda install package=version1
    安装或更新包，例如
    conda install numpy=1.9.3
    conda update numpy=1.93
    【指定channel进行安装package】
    conda install --channel[-c] https://conda.anaconda.org/anaconda tensorflow-gpu=1.9.0
    【手动安装package】
    pip 安装本地包
    pip install ～/Downloads/package-name.whl
    conda 安装本地包
    conda install --use-local ~/Downloads/package-name.tar.bz2
    执行上述命令，会在当前的环境下安装package_name包。 若无法使用conda install命令进行安装时，则可以使用pip命令进行安装。值得一提的是，pip只是包管理器，它无法对环境进行管理，所以要想在指定的环境中使用pip安装包，需要先切换到指定环境中使用pip命令进行安装。pip命令格式如下。
    pip install package_name
    例如，使用pip命令安装名称为see的包，示例如下。pip install see
    【卸载包】
    如果要在指定的环境中卸载包，则可以在指定环境下使用remove命令进行移除，命令格式如下。
    conda remove --name env_name package_name
    例如，卸载Python3环境下的pandas包，示例命令如下。
    conda remove --name python3 pandas
    同样，如果要卸载当前环境中的包，可以直接使用remove命令进行卸载，命令格式如下。
    conda remove package_name
    【更新包】
    更新当前环境下所有的包，可使用如下命令完成。
    conda update --a11
    如果只想更新某个包或某些包，则直接在update命令的后面加上包名即可，多个包之间使用空格隔开，示例命令如下。
    conda update numpy
    conda update pandas numpy matplotlib
